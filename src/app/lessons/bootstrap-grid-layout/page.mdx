# Grid System & Layout

Το πιο διάσημο κομμάτι του Bootstrap είναι το **12-column grid**: responsive layout χωρίς να γράψεις media queries. Αυτό το μάθημα το καλύπτει σε βάθος — containers, rows, columns, breakpoints, offsets, gutters και auto-layout — μαζί με τα layout utilities.

## container → row → col

Το grid στήνεται σε τρία επίπεδα, και κάθε `row` χωρίζεται σε **12 στήλες**:

```html
<div class="container">
  <div class="row">
    <div class="col-12 col-md-6 col-lg-4">Στήλη A</div>
    <div class="col-12 col-md-6 col-lg-4">Στήλη B</div>
    <div class="col-12 col-md-6 col-lg-4">Στήλη C</div>
  </div>
</div>
```

- `col-12` → όλο το πλάτος (12/12) σε mobile → οι στήλες **στοιβάζονται**.
- `col-md-6` → μισό (6/12) από tablet → 2 ανά γραμμή.
- `col-lg-4` → τρίτο (4/12) σε desktop → 3 ανά γραμμή.

**Κανόνας:** το άθροισμα των στηλών ανά row είναι **12**. Αν ξεπεράσει, τυλίγονται στην επόμενη γραμμή.

## Breakpoints

Τα responsive suffixes του Bootstrap:

| Suffix | Από πλάτος | Συσκευή |
|---|---|---|
| (καμία) | < 576 | extra small (mobile) |
| `sm` | ≥ 576px | small |
| `md` | ≥ 768px | tablet |
| `lg` | ≥ 992px | desktop |
| `xl` | ≥ 1200px | large |
| `xxl` | ≥ 1400px | extra large |

Όπως το Tailwind, είναι **mobile-first**: το `col-md-6` ισχύει από `md` **κι άνω**.

## Containers

- `.container` → fixed-width, με max-width ανά breakpoint (κεντραρισμένο περιεχόμενο).
- `.container-fluid` → 100% πλάτος πάντα.
- `.container-md` → fluid μέχρι το `md`, μετά fixed.

## Auto-layout, offsets & gutters

```html
<!-- auto-layout: ίσες στήλες χωρίς αριθμό -->
<div class="row"><div class="col">A</div><div class="col">B</div></div>

<!-- offset: άφησε κενό αριστερά (2 στήλες) -->
<div class="col-md-4 offset-md-2">Μετατοπισμένη</div>

<!-- gutters: έλεγχος κενού μεταξύ στηλών -->
<div class="row g-4">...</div>   <!-- g-4 = μεγαλύτερο gap -->
```

`col` (χωρίς αριθμό) μοιράζει ίσα τον χώρο· `offset-*` αφήνει κενό· `g-*` ελέγχει το gutter (κενό) ανάμεσα στις στήλες.

## Layout utilities

Πέρα από το grid, το Bootstrap έχει utilities για γρήγορες προσαρμογές:

```html
<div class="d-flex justify-content-between align-items-center">...</div>
<div class="d-none d-md-block">Κρυφό σε mobile, ορατό από md</div>
```

`d-flex` (display flex), `justify-content-*`, `align-items-*`, `d-none`/`d-block` (display, responsive) — συμπληρώνουν το grid για fine-tuning.

## Πώς απαντάς στη συνέντευξη

> «Explain Bootstrap's grid.»

> It's a 12-column responsive grid built as `container → row → col`. Each row is divided into 12 units, and I assign widths per breakpoint with classes like `col-12 col-md-6 col-lg-4` — full width and stacked on mobile, half from 768px, a third from 992px. The columns in a row should sum to 12; anything over wraps to the next line. It's mobile-first, so `col-md-6` applies from `md` up. I can use bare `col` for equal auto-layout columns, `offset-*` to push columns over, and `g-*` to control gutters — all without writing a single media query.
