# JavaScript Basics: κάνε τη σελίδα διαδραστική

Η HTML δίνει δομή και η CSS δίνει εμφάνιση. Η **JavaScript** δίνει συμπεριφορά: τι γίνεται όταν ο χρήστης πατάει ένα κουμπί, γράφει σε ένα input ή φορτώνει δεδομένα.

Σε αυτό το μάθημα θα χτίσεις έναν μικρό counter. Είναι απλό παράδειγμα, αλλά περιέχει τρεις ιδέες που θα χρησιμοποιείς συνέχεια:

- **Variable**: κρατάμε μια τιμή στη μνήμη.
- **Function**: βάζουμε λογική σε επαναχρησιμοποιήσιμο κομμάτι.
- **Event listener**: λέμε στον browser τι να κάνει όταν συμβεί ένα event, όπως click.

## 1. Variable

Μια variable είναι ένα όνομα που κρατά μια τιμή.

```js
let count = 0;
```

Εδώ λέμε ότι ο μετρητής ξεκινά από το `0`.

## 2. Function

Μια function είναι ένα μικρό block λογικής που μπορούμε να καλέσουμε όταν το χρειαζόμαστε.

```js
function increaseCount() {
  count = count + 1;
}
```

Αυτό αυξάνει το `count` κατά ένα.

## 3. Event listener

Με ένα event listener συνδέουμε τη function με μια ενέργεια του χρήστη.

```js
document.querySelector("#increase").addEventListener("click", increaseCount);
```

Αυτό σημαίνει: βρες το κουμπί με id `increase` και όταν γίνει click, τρέξε τη function `increaseCount`.
