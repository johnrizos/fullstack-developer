# CSS Box Model

Το box model είναι ο τρόπος που η CSS υπολογίζει τον χώρο κάθε element.

Κάθε element έχει:

- **content**: το πραγματικό περιεχόμενο.
- **padding**: εσωτερικό κενό.
- **border**: γραμμή γύρω από το element.
- **margin**: εξωτερικό κενό.

Αν δεν καταλάβεις box model, τα layouts θα μοιάζουν τυχαία. Αν το καταλάβεις, οι αποστάσεις γίνονται ελεγχόμενες.
