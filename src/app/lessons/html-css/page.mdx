# HTML & CSS: χτίζοντας την πρώτη σελίδα

Στο προηγούμενο μάθημα είδες πώς ταξιδεύει μια ιστοσελίδα από τον server στον browser. Τώρα θα δούμε από τι αποτελείται η σελίδα που τελικά εμφανίζεται.

Το μάθημα έχει τρία βήματα: καταλαβαίνεις τη βασική ιδέα, πειράζεις κώδικα, και μετά περνάς μια μικρή αποστολή με αυτόματο checklist.

## 1. Η HTML δίνει δομή

Η **HTML** (HyperText Markup Language) περιγράφει τη δομή της σελίδας. Με απλά λόγια, λέει στον browser τι είναι τίτλος, τι είναι παράγραφος, τι είναι κουμπί και ούτω καθεξής.

Τα στοιχεία της HTML γράφονται με tags:

- Για μεγάλο τίτλο χρησιμοποιούμε `<h1>Κείμενο τίτλου</h1>`.
- Για παράγραφο χρησιμοποιούμε `<p>Κείμενο παραγράφου</p>`.
- Για κουμπί χρησιμοποιούμε `<button>Κλικ εδώ</button>`.

Δοκίμασε πρώτα να αλλάξεις το κείμενο μέσα στον editor. Το αποτέλεσμα ενημερώνεται live.

<LiveEditor

  files={example1}
/>

## 2. Η CSS δίνει εμφάνιση

Αν η HTML είναι η δομή, η **CSS** (Cascading Style Sheets) είναι η εμφάνιση: χρώματα, αποστάσεις, μεγέθη, στοίχιση και γενικότερα το πώς φαίνεται η σελίδα.

Στην CSS επιλέγουμε ένα HTML element και του δίνουμε κανόνες.

```css
h1 {
  color: blue;
}
```

Αυτό σημαίνει: βρες όλα τα `h1` και κάνε το χρώμα του κειμένου μπλε.

## 3. Mini project: προσωπική κάρτα

Τώρα δεν αρκεί να διαβάσεις. Φτιάξε μια μικρή profile card και πέρασε τα κριτήρια. Το checklist στα δεξιά θα ενημερώνεται όσο γράφεις.

Η αποστολή σου:

1. Κράτα ένα `h1`, ένα `p` και ένα `button` στην HTML.
2. Βάλε το όνομά σου ή ένα ψευδώνυμο στον τίτλο.
3. Άλλαξε το `color` του `h1` σε `red`.
4. Άλλαξε το `background-color` του `button` σε `blue`.
