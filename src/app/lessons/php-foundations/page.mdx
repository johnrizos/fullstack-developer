# PHP Foundations

Η PHP τρέχει ένα τεράστιο κομμάτι του web (WordPress, Laravel, Symfony). Η μοντέρνα PHP 8 δεν έχει σχέση με τη φήμη της παλιάς PHP: έχει τύπους, JIT, enums και καθαρό OOP.

Κάθε αρχείο PHP ξεκινά με `<?php`. Ο κώδικας τρέχει στον server και επιστρέφει HTML/JSON στον browser.

```php
<?php
echo "Hello, world!";
```

## Variables & τύποι

Οι μεταβλητές ξεκινούν με `$`. Η PHP είναι **dynamically typed** αλλά **strongly typed** στις πράξεις.

```php
<?php
$name = "Άννα";        // string
$age = 30;              // int
$price = 9.99;          // float
$active = true;         // bool
$tags = ["php", "web"]; // array
$nothing = null;        // null
```

Δες τον τύπο με `gettype()` ή `var_dump()` (το `var_dump` δείχνει τύπο + τιμή — το βασικό σου debugging tool).

```php
<?php
var_dump($age); // int(30)
```

## strict_types — γράψε σαν professional

Από προεπιλογή η PHP κάνει «χαλαρό» coercion (`"5" + 5 === 10`). Σε σοβαρό κώδικα βάζεις στην κορυφή κάθε αρχείου:

```php
<?php
declare(strict_types=1);
```

Με `strict_types`, αν μια function περιμένει `int` και της δώσεις `"5"`, πετάει `TypeError` αντί να το «μαντέψει». Λιγότερα κρυφά bugs.

## String interpolation & concatenation

```php
<?php
$name = "Άννα";
echo "Γεια σου $name!";          // interpolation σε double quotes
echo 'Γεια σου $name!';          // single quotes → ΧΩΡΙΣ interpolation
echo "Σύνολο: {$price} €";       // {} για σαφήνεια
echo "Γεια " . $name . "!";      // concatenation με τελεία (.)
```

Προσοχή: η PHP ενώνει strings με **τελεία `.`**, όχι με `+` (το `+` είναι μόνο αριθμητικό).

## Control flow

```php
<?php
$score = 75;

if ($score >= 50) {
    echo "Pass";
} elseif ($score >= 40) {
    echo "Borderline";
} else {
    echo "Fail";
}

// match (PHP 8): σαν switch αλλά επιστρέφει τιμή και κάνει strict (===) σύγκριση
$label = match (true) {
    $score >= 50 => "Pass",
    $score >= 40 => "Borderline",
    default      => "Fail",
};
```

Το `match` είναι το μοντέρνο `switch`: δεν χρειάζεται `break`, δεν κάνει fall-through, και συγκρίνει με `===`.

## Loops

```php
<?php
for ($i = 0; $i < 3; $i++) {
    echo $i;
}

$fruits = ["μήλο", "μπανάνα"];
foreach ($fruits as $fruit) {
    echo $fruit;
}

// associative
$user = ["name" => "Άννα", "age" => 30];
foreach ($user as $key => $value) {
    echo "$key: $value\n";
}
```

Το `foreach` είναι ο πιο συχνός loop στην PHP — δουλεύει και σε indexed και σε associative arrays.

## == vs === (η κλασική παγίδα)

```php
<?php
var_dump(0 == "a");    // PHP 8: false (σε PHP 7 ήταν true!)
var_dump("1" == 1);    // true  — loose, κάνει coercion
var_dump("1" === 1);   // false — strict, ΚΑΙ τύπος ΚΑΙ τιμή
```

Κανόνας: **χρησιμοποίησε πάντα `===`** εκτός αν ξέρεις πολύ καλά γιατί θες loose σύγκριση.

## Πώς απαντάς στη συνέντευξη

«What does modern PHP look like today?»

> Modern PHP 8 is typed and strict. I start files with `declare(strict_types=1)`, use type hints on parameters and return types, prefer `match` over `switch` for value-returning branches, and always compare with `===`. I rely on `var_dump` for quick inspection. The language has enums, readonly properties, named arguments and the nullsafe operator, so it reads much closer to typed languages than the old PHP reputation suggests.
