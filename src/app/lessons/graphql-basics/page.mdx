# GraphQL Basics

GraphQL είναι API style όπου ο client ζητά ακριβώς τα fields που χρειάζεται.

## Query

```graphql
query {
  tasks {
    id
    title
    completed
  }
}
```

## Mutation

```graphql
mutation {
  createTask(title: "Learn GraphQL") {
    id
    title
  }
}
```

## Schema

Το schema είναι contract.

```graphql
type Task {
  id: ID!
  title: String!
  completed: Boolean!
}
```

## REST vs GraphQL

REST είναι απλό και predictable για πολλά APIs. GraphQL βοηθά όταν έχεις πολλά clients ή σύνθετα data requirements και θέλεις να αποφύγεις overfetching/underfetching.

## Interview answer template
