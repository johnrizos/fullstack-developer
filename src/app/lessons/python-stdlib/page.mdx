# Standard Library Mastery

«Batteries included» είναι το motto της Python — και το να ξέρεις τη standard library είναι αυτό που ξεχωρίζει τον developer που γράφει 30 γραμμές χειροκίνητα από αυτόν που λύνει το ίδιο πρόβλημα με μία κλήση στο `collections` ή στο `itertools`. Στη συνέντευξη, το «πώς θα μετρούσες συχνότητες;» ή «πώς ομαδοποιείς;» απαντιέται σωστά με stdlib, όχι με reinventing the wheel.

## Τι πρέπει να μπορείς στο τέλος

- να αξιοποιείς το `collections` (`Counter`, `defaultdict`, `deque`, `namedtuple`),
- να χρησιμοποιείς `functools` (`reduce`, `lru_cache`, `partial`),
- να συνδυάζεις `itertools` σε πραγματικά προβλήματα,
- να δουλεύεις paths με `pathlib` αντί για `os.path`,
- να χειρίζεσαι ημερομηνίες/ώρες σωστά με `datetime` (timezone-aware).

## `collections` — οι δομές που λείπουν

```py
from collections import Counter, defaultdict, deque

# Counter — μετράει συχνότητες αυτόματα
Counter("banana")               # Counter({'a': 3, 'n': 2, 'b': 1})
Counter(words).most_common(3)   # τα 3 πιο συχνά ως [(item, count), ...]

# defaultdict — δίνει default όταν λείπει το key (όχι KeyError)
groups = defaultdict(list)
for user in users:
    groups[user.role].append(user)   # χωρίς να ελέγχεις αν υπάρχει το key

# deque — double-ended queue, O(1) και στις δύο άκρες
q = deque([1, 2, 3])
q.appendleft(0)     # O(1) — σε list το insert(0, x) είναι O(n)!
q.popleft()         # O(1) — ideal για queues/BFS
```

Τρία idioms που πρέπει να αναγνωρίζεις αμέσως:
- **Μέτρημα** → `Counter`, όχι χειροκίνητο dict με `.get`.
- **Ομαδοποίηση** → `defaultdict(list)`, όχι `if key not in d: d[key] = []`.
- **Queue/FIFO** → `deque`, όχι `list.pop(0)` (που είναι O(n)).

## `functools` — εργαλεία για functions

```py
from functools import reduce, lru_cache, partial

# reduce — αναδίπλωση σε μία τιμή
reduce(lambda acc, x: acc + x, [1, 2, 3, 4], 0)   # 10 (αλλά sum() καλύτερο εδώ)
reduce(lambda acc, x: acc * x, [1, 2, 3, 4], 1)   # 24 (factorial-style)

# lru_cache — memoization δωρεάν
@lru_cache(maxsize=None)
def fib(n: int) -> int:
    return n if n < 2 else fib(n - 1) + fib(n - 2)
fib(100)    # στιγμιαίο — χωρίς cache θα ήταν εκθετικά αργό

# partial — "καρφώνει" arguments, φτιάχνει νέα function
from functools import partial
basetwo = partial(int, base=2)
basetwo("1010")    # 10 — int(x, base=2) με το base προ-συμπληρωμένο
```

Το `lru_cache` είναι το πιο πρακτικό: μετατρέπει εκθετική αναδρομή σε γραμμική, ή cache-άρει ακριβές pure functions, με μία γραμμή.

## `itertools` σε πραγματικά προβλήματα

```py
import itertools as it

# Ομαδοποίηση ΔΙΑΔΟΧΙΚΩΝ (χρειάζεται sorted πρώτα!)
data = sorted(items, key=lambda x: x.category)
for category, group in it.groupby(data, key=lambda x: x.category):
    print(category, list(group))

# Pairwise — διαδοχικά ζευγάρια (3.10+)
list(it.pairwise([1, 2, 3, 4]))     # [(1,2),(2,3),(3,4)] — ideal για diffs

# Accumulate — running totals
list(it.accumulate([1, 2, 3, 4]))   # [1, 3, 6, 10]

# Combinations / permutations
list(it.combinations("ABC", 2))     # [('A','B'),('A','C'),('B','C')]
```

> Trap στο `groupby`: ομαδοποιεί μόνο **διαδοχικά** ίσα στοιχεία — αν τα δεδομένα δεν είναι ταξινομημένα κατά το key, θα πάρεις σπασμένες ομάδες. Πάντα `sorted` με το ίδιο key πρώτα. Κλασικό σημείο όπου χάνονται άνθρωποι.

## `pathlib` — paths σαν objects

Ξέχνα το `os.path.join` με strings — το `pathlib` δίνει καθαρό, cross-platform API:

```py
from pathlib import Path

root = Path("data")
config = root / "settings" / "app.json"   # το / κάνει join, cross-platform!

config.exists()
config.suffix          # '.json'
config.stem            # 'app'
config.read_text()     # διάβασμα σε μία γραμμή
list(root.glob("**/*.py"))   # όλα τα .py αναδρομικά
Path("out").mkdir(parents=True, exist_ok=True)
```

Το `/` operator για join είναι το σήμα κατατεθέν — διαβάζεται φυσικά και δουλεύει σε Windows/Linux χωρίς `\` vs `/` προβλήματα.

## `datetime` — με timezone, πάντα

Το πιο συχνό production bug με χρόνο: **naive datetimes** (χωρίς timezone). Σε servers, δούλευε πάντα timezone-aware σε UTC:

```py
from datetime import datetime, timezone, timedelta

datetime.now(timezone.utc)          # ✓ aware, UTC — ο σωστός τρόπος σε server
datetime.now()                      # ✗ naive — άγνωστη ζώνη, πηγή bugs

deadline = datetime.now(timezone.utc) + timedelta(days=7)
iso = deadline.isoformat()          # serialize για API/DB
datetime.fromisoformat(iso)         # parse πίσω
```

Ο κανόνας: **store & compute σε UTC, convert σε local μόνο για display**. Naive datetimes που συγκρίνονται με aware πετάνε `TypeError` — γι' αυτό η συνέπεια έχει σημασία.

## Πώς απαντάς στη συνέντευξη

«How would you count word frequencies and return the top three?»

> `Counter(words).most_common(3)` — the `collections` module handles it in one call and returns `(word, count)` tuples sorted by frequency, so there's no manual dict bookkeeping. More broadly, I reach for the standard library before writing loops: `Counter` for frequencies, `defaultdict(list)` for grouping, `deque` for O(1) queues, `lru_cache` for memoization, `itertools.groupby`/`pairwise`/`accumulate` for sequence work, `pathlib` for filesystem paths, and timezone-aware `datetime` in UTC for anything time-related. Knowing the batteries that ship with Python turns a lot of twenty-line problems into one expressive line.
