# Docker & CI/CD

Docker και CI/CD εμφανίζονται συχνά σε αγγελίες γιατί δείχνουν ότι μπορείς να παραδώσεις λογισμικό με προβλέψιμο τρόπο.

## Docker

Docker container πακετάρει app + dependencies.

```dockerfile
FROM node:20-alpine
WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
RUN npm run build
CMD ["npm", "start"]
```

## CI

Continuous Integration σημαίνει ότι checks τρέχουν αυτόματα σε κάθε PR.

```yaml
- run: npm ci
- run: npm run lint
- run: npm run build
```

## CD

Continuous Deployment/Delivery σημαίνει ότι verified αλλαγές μπορούν να πάνε σε environment με προβλέψιμο workflow.
