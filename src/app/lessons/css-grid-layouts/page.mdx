# CSS Grid & Page Layouts

Το CSS Grid είναι εργαλείο για layout σε δύο διαστάσεις: columns και rows. Το χρησιμοποιείς σε dashboards, galleries, admin panels, pricing sections και card layouts.

Αν το Flexbox είναι καλό για μία γραμμή ή μία στήλη, το Grid είναι καλό όταν θέλεις ολόκληρη δομή σελίδας.

## Grid container

Με `display: grid` ένα element γίνεται grid container.

```css
.layout {
  display: grid;
  gap: 24px;
}
```

Το `gap` βάζει απόσταση ανάμεσα στα grid items, χωρίς να χρειάζεται margin σε κάθε παιδί.

## Columns

Με `grid-template-columns` ορίζεις τις στήλες.

```css
.dashboard {
  display: grid;
  grid-template-columns: 220px 1fr;
}
```

Αυτό σημαίνει:

- πρώτη στήλη: 220px,
- δεύτερη στήλη: όλος ο υπόλοιπος χώρος.

Το `fr` σημαίνει fraction του διαθέσιμου χώρου.

## repeat και responsive cards

Για card grids χρησιμοποιείς συχνά `repeat`.

```css
.cards {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 16px;
}
```

Αυτό φτιάχνει 3 ίσες στήλες.

Για πιο flexible layout:

```css
.cards {
  grid-template-columns: repeat(auto-fit, minmax(220px, 1fr));
}
```

Το `minmax(220px, 1fr)` λέει: κάθε card να έχει τουλάχιστον 220px, αλλά να μπορεί να μεγαλώσει.

## Grid vs Flexbox

Χρησιμοποίησε Flexbox όταν:

- έχεις navbar,
- έχεις toolbar,
- θέλεις items σε μία σειρά,
- θες εύκολο align center.

Χρησιμοποίησε Grid όταν:

- έχεις sidebar + main content,
- έχεις cards σε rows/columns,
- χτίζεις gallery,
- θέλεις page layout.

Στην πράξη τα χρησιμοποιείς μαζί. Grid για τη μεγάλη δομή, Flexbox για μικρά controls μέσα στα sections.

## Responsive Grid

Σε μικρές οθόνες, ένα dashboard με sidebar και content πρέπει συχνά να γίνει μία στήλη.

```css
@media (max-width: 700px) {
  .dashboard {
    grid-template-columns: 1fr;
  }
}
```

Αυτό είναι job-ready skill: δεν αρκεί να φαίνεται καλό σε desktop. Πρέπει να μη σπάει σε mobile.

- Μπορείς να φτιάξεις cards grid.
