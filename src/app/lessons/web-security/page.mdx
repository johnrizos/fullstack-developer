# Web Security Basics

Security δεν είναι “θα το δούμε στο τέλος”. Κάθε fullstack developer πρέπει να ξέρει τα βασικά.

## XSS

Cross-site scripting συμβαίνει όταν untrusted input εκτελείται σαν JavaScript στο browser.

Άμυνες:

- escaping/sanitization,
- όχι raw HTML από χρήστη,
- Content Security Policy,
- frameworks που κάνουν escaping by default.

## CSRF

CSRF είναι όταν τρίτο site προσπαθεί να κάνει request με τα cookies του χρήστη.

Άμυνες:

- SameSite cookies,
- CSRF tokens,
- έλεγχος origin/referer σε sensitive actions.

## Validation

Πρέπει να κάνεις validation server-side. Client-side validation είναι για UX, όχι για security.

## Secrets

API keys, database URLs, OAuth secrets:

- όχι στο frontend bundle,
- όχι σε Git,
- χρήση environment variables/secret manager.
