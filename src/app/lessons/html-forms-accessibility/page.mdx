# HTML Forms & Accessibility

Οι φόρμες είναι ο τρόπος που ο χρήστης στέλνει δεδομένα: login, αναζήτηση, checkout, contact form.

Το βασικό για σωστή φόρμα είναι να συνδέεις κάθε input με label. Το `for` του label πρέπει να ταιριάζει με το `id` του input.

```html
<label for="email">Email</label>
<input id="email" type="email" />
```

Αυτό βοηθά και στην προσβασιμότητα και στη χρηστικότητα.
