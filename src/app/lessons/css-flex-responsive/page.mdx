# CSS Flexbox & Responsive Layouts

Το Flexbox είναι από τα πιο χρήσιμα εργαλεία για layout. Το χρησιμοποιείς για navbars, cards, toolbars και σειρές από στοιχεία.

Βασικές ιδιότητες:

- `display: flex`: ενεργοποιεί flex layout.
- `gap`: βάζει απόσταση ανάμεσα στα στοιχεία.
- `flex-wrap: wrap`: επιτρέπει στα στοιχεία να πάνε σε νέα γραμμή όταν δεν χωράνε.
- `justify-content`: ελέγχει την οριζόντια στοίχιση.
