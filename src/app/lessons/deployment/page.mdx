# Deployment & Cloud

Deployment σημαίνει ότι η εφαρμογή τρέχει για πραγματικούς χρήστες, όχι μόνο στο laptop σου.

## Build

Πριν deploy, πρέπει να περνάει:

```bash
npm run lint
npm run build
```

## Environment variables

Τα env vars κρατούν configuration:

- database URLs,
- API keys,
- OAuth secrets,
- feature flags.

Δεν μπαίνουν στο Git.

## Vercel / AWS / Azure / GCP

Για frontend/Next.js συχνά βλέπεις Vercel. Για πιο enterprise fullstack, συχνά AWS/Azure/GCP.

Βασικές έννοιες:

- hosting,
- serverless functions,
- object storage,
- managed databases,
- logs.
